## Tentang Program Ini

Program ini hanya sebuah dashboard aplikasi ticket yang dibuat dengan sebuah framework laravel 7, didalamnya terdapat beberapa fitur antara lain:

- Fitur Login
- Fitur Logout
- Fitur Register / Create
- Fitur Search Query
- Fitur Update.
- Fitur Delete.
